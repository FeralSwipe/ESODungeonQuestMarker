# ESO Dungeon Quest Marker

Аддон для The Elder Scrolls Online, который помогает находить групповые подземелья с невыполненным заданием на очко навыка.

В меню **«Поиск подземелий → Конкретные подземелья»** рядом с названием подземелья появляется значок задания `!`, если текущий персонаж ещё не выполнил его сюжетное задание и не получил соответствующее очко навыка.

## Возможности

- Проверяет прохождение отдельно для текущего персонажа.
- Показывает отметку возле нормальной и ветеранской версии подземелья.
- Не показывает отметки для DLC-подземелий, к которым у аккаунта нет доступа.
- Учитывает как купленные DLC, так и временный доступ через ESO Plus.
- Работает независимо от Dungeon Tracker и не требует сторонних библиотек.
- Использует внутренние идентификаторы подземелий, поэтому не зависит от языка клиента.
- Совместим с PledgeHighlights: значки двух аддонов располагаются рядом и не перекрывают друг друга.

## Установка

1. Скачайте репозиторий.
2. Скопируйте папку `DungeonQuestMarker` в:

   ```text
   Documents/Elder Scrolls Online/live/AddOns
   ```

3. Включите **Dungeon Quest Marker** в меню модификаций ESO.
4. Выполните команду `/reloadui` или перезапустите игру.

## Использование

Откройте **«Поиск группы и занятий → Поиск подземелий → Конкретные подземелья»**. Значок `!` отображается только возле доступных подземелий, где задание на очко навыка ещё не выполнено текущим персонажем.

После завершения задания отметка обновится автоматически.

## Совместимость

- Версия аддона: `1.0.1`
